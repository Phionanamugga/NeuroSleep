🧠 NeuroSleep App
Track, visualize, and pay off your sleep debt with science-backed insights.
NeuroSleep is an intelligent sleep debt tracking app designed to help you understand, manage, and repay your "sleep debt." By combining neuroscience, smart analytics, and user-friendly design, NeuroSleep empowers you to take control of your sleep health and boost your daily performance.

🌙 Key Features
✅ Sleep Debt Calculator – Know exactly how much sleep you owe your body.
📊 Intuitive Dashboard – Track your weekly trends, sleep efficiency, and circadian patterns.
🔔 Personalized Reminders – Smart notifications to help you catch up on sleep.
📈 Analytics – View correlations between sleep debt and mood, productivity, or activity.
🔒 Privacy First – Your data stays on your device (or securely stored).

🚀 Quick Start
1. Clone the repository
git clone https://github.com/yourusername/neurosleep.git
cd neurosleep
2. Install dependencies
pip install -r requirements.txt
3. Run the app
uvicorn main:app --reload
Open your browser and navigate to http://127.0.0.1:8000
